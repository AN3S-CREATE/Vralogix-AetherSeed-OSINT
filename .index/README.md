# `.index/` — project context store

Living, queryable context for humans and coding agents working on **AetherSeed**.

## Files

| File | Purpose |
|------|---------|
| `README.md` | This guide |
| `file-inventory.md` | Key files + one-line purpose + status |
| `architecture.md` | High-level system overview |
| `key-decisions.md` | Lightweight ADR-style decisions |
| `dead-code.md` | Unused/legacy code notes |
| `context-refresh-log.md` | Index refresh history |

## Maintenance

- Update inventory after create/edit/delete of material files.
- Update architecture/decisions when structure or trade-offs change.
- Prefer concise tables; never store secrets.
- Agents: read this folder before starting work; refresh after changes.
